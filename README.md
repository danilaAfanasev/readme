# Знакомство с Git

Git – это удобная технология позволяющая объединить работу нескольких сотрудников в одном месте. Каждый видит изменения в проекте и видил что делают другие. 

## Команды

Создание пустого репозитория в папке:
```
git init
```

Показать текущую ситуацию по локальному репозиторию:
```
git status
```

Добавить файл:
```
git add
```

Сделать коммит:
```
git commit -m ""
```
